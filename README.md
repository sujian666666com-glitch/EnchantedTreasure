## 仓库简介

`EnchantedTreasure` 是魔仙小队的资源共享中心，用于沉淀和分发团队协作过程中积累的各类资产。

仓库主要覆盖：
- **文档资料**：项目文档、会议纪要、学习笔记、最佳实践
- **设计素材**：图标、模板、配色方案、UI 资源
- **代码片段**：常用工具函数、脚本、配置模板
- **知识沉淀**：技术分享、问题解决方案、经验总结

## 贡献指南

### 命名规范

- **文档文件**：使用清晰的中文或英文命名，如 `会议纪要-20260330.md`
- **设计素材**：按用途分类，如 `icons/`、`images/`、`templates/`
- **代码片段**：附带简短说明，如 `部署脚本/`、`常用函数/`

### 提交建议

1. 提交前确认文件分类正确
2. 大文件请考虑压缩或使用外部链接
3. 敏感信息（密码、密钥等）禁止提交
4. 提交信息使用规范格式：
   - `Add: 新增资源`
   - `Update: 更新资源`
   - `Fix: 修正错误`
   - `Remove: 删除资源`
